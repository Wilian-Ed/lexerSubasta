Para realizar una puja:

Debe ingresar en el siguiente cuadro de texto su
nombre de usuario, el valor que desea pujar y su
código. ¡Usted puede realizar más de una puja!

Para ser candidato debe seguir estas reglas:

1. Su nombre debe iniciar obligatoriamente con una
letra, seguidamente puede optar por una sucesión
de letras, dígitos o símbolos de subrayado (_)
2. El valor solo puede contener una única cifra
decimal. Ejemplo: 0.5 ó 0.0
3. El código comienza por el símbolo de tanto por
ciento (%) al que le siguen tres letras o un
múltiplo de tres letras (pero nunca espacios).
